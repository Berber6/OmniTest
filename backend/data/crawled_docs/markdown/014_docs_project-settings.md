# Project Settings
If you are manager of the project that you are currently editing, click the "cog in the box" icon in the top-right corner next to `+ Add Board` button to access this project settings.
![Project settings buttons](https://docs.4gaboards.com/docs/project-settings/)
Alternatively you can use ellipsis menu from the sidebar to access settings to any of your managed projects. Just hover over the desired project name and click on ellipsis icon.
![Project context menu](https://docs.4gaboards.com/assets/images/projectsmenu_en-74909a67e4470a6550d30825e7bd4ac3.png)
When you choose the project you have options to change the title (click `Save` button to save changes),add or remove managers from the project and set the background image (with possibility to upload your own). The image will be visible in [projects view](https://docs.4gaboards.com/docs/project). For extra caution, the `Delete Project` button was situated in the `Danger Zone`. And no worries, we added an additional pop-up for confirmation.
![Project settings view](https://docs.4gaboards.com/assets/images/projectsettings_en-a1445c6c2ffa02d3c349f1d4449b57bf.png)
To go back to the project, click on arrow icon in the top-right corner or click the 4ga Boards logo to access [projects view](https://docs.4gaboards.com/docs/project).
